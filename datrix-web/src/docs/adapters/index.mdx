# Adapters

Adapters translate Datrix's database-agnostic `QueryObject` into database-specific
queries. Pass an adapter instance to `adapter` in `defineConfig()`.

---

| Adapter | Description |
| --- | --- |
| [PostgreSQL](/docs/adapters/postgres) | Production-ready. Full SQL support including transactions and migrations. |
| [MySQL](/docs/adapters/mysql) | Production-ready. Works with MySQL 8+ and MariaDB 10.5+. |
| [MongoDB](/docs/adapters/mongodb) | Requires a replica set for transactions. |
| [JSON](/docs/adapters/json) | File-based. For development and testing only. |
