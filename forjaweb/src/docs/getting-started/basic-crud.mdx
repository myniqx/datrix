# Basic CRUD

Once your config and schemas are in place, import the factory and start querying.

```typescript
import getForja from './forja.config'

const forja = await getForja()
```

## Create

```typescript
const user = await forja.create('user', {
  name: 'Alice',
  email: 'alice@example.com',
  role: 'editor',
})
// → { id: 1, name: 'Alice', email: 'alice@example.com', role: 'editor', createdAt: ..., updatedAt: ... }
```

## Read

```typescript
// Single record by id
const user = await forja.findById('user', 1)

// Single record by condition
const user = await forja.findOne('user', { email: 'alice@example.com' })

// Multiple records
const users = await forja.findMany('user', {
  where: { role: 'editor' },
  sort: { createdAt: 'desc' },
  limit: 20,
  offset: 0,
})

// Count
const total = await forja.count('user', { role: 'editor' })
```

## Update

```typescript
// Update by id
const updated = await forja.update('user', 1, { name: 'Alice Smith' })

// Update multiple records matching a condition
const updated = await forja.updateMany('user', { role: 'editor' }, { role: 'user' })
```

## Delete

```typescript
// Delete by id
const deleted = await forja.delete('user', 1)

// Delete multiple records matching a condition
const deleted = await forja.deleteMany('user', { role: 'user' })
```

## Bulk create

```typescript
const users = await forja.createMany('user', [
  { name: 'Bob',   email: 'bob@example.com' },
  { name: 'Carol', email: 'carol@example.com' },
])
```

For filtering, sorting, pagination and relation population details see [Core: Query](/docs/core/query).
