# Plugins

Plugins extend Forja's core functionality. They can add new schemas, extend existing ones, intercept queries, and integrate with external services.

---

## Official plugins

| Plugin  | Package      | Description                                                             |
| ------- | ------------ | ----------------------------------------------------------------------- |
| **API** | `@forja/api` | REST API layer with automatic CRUD routes and optional JWT/session auth |
